## architecture

youtube-clone/
├── index.html            # Homepage (video listings)
├── video.html            # Video player page
├── channel.html          # Channel page (optional)
│
├── assets/               # All static files
│   ├── images/           # Logos, icons, buttons
│   ├── thumbnails/       # Video thumbnails
│   └── videos/           # Sample videos (if needed)
│
├── css/
│   ├── main.css          # Global styles (fonts, colors)
│   ├── header.css        # Header specific styles
│   ├── video-card.css    # Video listing styles
│   └── player.css        # Video player page styles
│
└── js/
    ├── main.js           # Shared functions
    ├── home.js           # Homepage functionality
    ├── player.js         # Video player functionality
    └── helpers.js        # Utility functions
